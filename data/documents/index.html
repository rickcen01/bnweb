注意： 这些查看器使用 Mozilla 的 PDF.js 库直接在浏览器中渲染 PDF 文件，无需插件。所有处理都在您的浏览器中本地进行 - 不会将任何文件上传到任何服务器。

示例 PDF

没有可用的 PDF 文件？您可以使用我们的示例 HTML 页面生成一个用于测试的 PDF。

查看示例 HTML

从示例页面创建 PDF：

  1. 使用上面的链接打开示例 HTML 页面
  2. 使用浏览器的打印功能（Ctrl+P 或 Cmd+P）
  3. 选择"另存为 PDF"作为目标
  4. 点击"保存"并选择计算机上的位置
  5. 将保存的 PDF 上传到其中一个查看器
